# 🛍️ Online Store Homepage – Day 4 Report

## 📅 Day 4 Progress

### 🔹 Work Done

On Day 4, I focused on implementing a complete **shopping cart system** in the RoyalMart project. The main goal was to make the website function like a real e-commerce platform by allowing users to add, view, and manage products in the cart.

---

### 🔹 Features Implemented

* Developed a **dynamic cart system** using JavaScript
* Enabled **Add to Cart functionality with product details**:

  * Product Name
  * Price
  * Image
* Stored cart data using **localStorage** for persistence
* Created a **cart page (cart.html)** to display added products
* Displayed **all cart items dynamically**
* Implemented **total price calculation** for all items
* Added **Remove button** to delete items from the cart
* Updated cart count dynamically based on number of items

---

### 🔹 Technologies Used

* HTML
* CSS
* JavaScript

---

### 🔹 Outcome

Successfully built a fully functional cart system where users can add products, view them in a dedicated cart page, see total pricing, and remove items. This significantly improved the project by adding real-world e-commerce functionality.
